# Team081-cs411114

## To run the flask backend server, we first setup the venv:
`cd server`
`python -m venv .env`
### Activate the venv:
#### Mac:
`source .env/bin/activate`
#### Windows:
`.env\Scripts\activate`
### Install packages:
`pip install -r requirements.txt`
### Run the backend:
`python -m flask --app app run`

## To run the frontend:
`cd ../frontend`
`npm install`
`npm start`
